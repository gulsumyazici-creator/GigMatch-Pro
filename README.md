# GigMatch Pro

GigMatch Pro is a Java-based freelancing platform simulation developed as a Data Structures and Algorithms course project. The system manages customers, freelancers, job requests, ratings, cancellations, blacklist operations, loyalty tiers, burnout status, service changes, and monthly simulations.

The project focuses on efficient data management and freelancer ranking for large-scale input processing. Freelancers are matched to customer requests using a composite score based on skill compatibility, average rating, reliability, and burnout status.

## Features

* Customer and freelancer registration
* Direct freelancer employment
* Automatic job request matching
* Composite freelancer ranking system
* Rating and skill update mechanism
* Customer and freelancer cancellations
* Personal and platform-level blacklist handling
* Monthly simulation with burnout, recovery, loyalty tier updates, and service changes
* File-based input and output processing

## Technologies

* Java
* Object-Oriented Programming
* Data Structures and Algorithms
* File I/O

## How to Run

Compile the source files:

```bash
javac *.java
```

Run the program:

```bash
java Main <input_file> <output_file>
```
